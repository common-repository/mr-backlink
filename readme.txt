== Installation ==

1. Upload Mr Backlink plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Select the plugin mr backlink from the admin menu in your dashboard
3. Select the category your website belongs to 
4. Write a good looking keyword that describes your website content/service 
5. Click save to see your website status and wait for the approval , Please don't submit your website in case it is in maintenance mode or under construction as it will be manually checked .
